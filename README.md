# Esperança Viva – Plataforma Institucional para ONGs

Site desenvolvido em **HTML, CSS e JavaScript**, que apresenta a ONG fictícia **Esperança Viva**, seus projetos sociais e oferece meios de contato para voluntários e doadores. O projeto é responsivo e utiliza uma paleta de cores azul e branca.

---

## Funcionalidades
- Exibição de informações institucionais da ONG fictícia.
- Página de projetos sociais com detalhes.
- Formulário de cadastro de voluntários.
- Navegação responsiva para desktop e mobile.  
- Destaque da página ativa no menu.
- Rolagem suave para âncoras internas.  

---

## Tecnologias Utilizadas
- [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML) — estrutura do site  
- [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS) — estilização das páginas  
- [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — funcionalidades interativas
  
---

## Como Funciona
1. **Home** → página inicial com apresentação da ONG e destaques principais.  
2. **Projetos** → lista de projetos sociais com descrições e imagens.  
3. **Cadastro de Voluntário** → formulário com validação para registro de novos voluntários.  
4. **Contato** → formulário de contato e informações de comunicação com a ONG.  
5. **Menu responsivo** → navegação adaptável para dispositivos móveis.
